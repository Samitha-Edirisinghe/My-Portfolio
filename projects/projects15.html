MR Simulator Android Project

Category : Augmented Reality Mobile Application

Client : My university Project

Description : This project involved the development of a Mixed Reality (MR) Simulator, an Android mobile application designed to showcase the capabilities of Augmented Reality (AR). The application allows users to place virtual objects—in this case, a 3D panda model—into their real-world environment using the device's camera. The simulator is a proof-of-concept for how augmented reality can blend digital content with the physical world, creating interactive and immersive experiences. The primary goal was to demonstrate proficiency in mobile application development and the integration of AR technologies. The application provides a simple yet effective user interface to interact with the virtual object, which can be seen in three distinct camera views.

Technologies : Java, Kotlin, ARCore, Sceneform for rendering the 3D model.
