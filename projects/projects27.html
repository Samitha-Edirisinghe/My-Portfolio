Ui/UX Design

GuardianEye Mobile Figma

Category : Ui/UX Design

Client : My FYP Project

Description : This project showcases the UI/UX design for GuardianEye, a mobile application designed to be a companion for a CCTV Management System. The application allows users to monitor real-time crime data, including location, time, status, and video feeds, all from a mobile device. The design, created entirely in Figma, focuses on creating a dark-themed, modern, and highly intuitive interface that presents critical security information in a clear and digestible format. This project highlights a strong understanding of mobile user experience principles, information hierarchy, and visual design.

Technologies : Figma
